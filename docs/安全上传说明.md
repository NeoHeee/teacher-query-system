# 安全上传说明

本仓库是公开仓库，只适合放可公开下载的 Windows 单文件版程序和说明文档。

不要上传以下内容：

- `data/teacher_system.db` 数据库文件
- `data/attachments` 附件目录
- `data/auto_backups` 自动备份目录
- 激活码生成器
- 激活私钥
- 客户真实数据
- 用户账号、密码、联系方式等敏感信息

建议公开上传内容：

- `README.md`
- `CHECKSUMS.txt`
- `师资信息查询系统-Windows单文件版.exe`

如果需要发布新版，建议流程如下：

1. 先在本地确认程序能正常启动。
2. 确认登录页没有默认填入账号密码。
3. 确认不会带上本地 `data` 文件夹。
4. 计算 exe 的 SHA256 校验值。
5. 上传新的 exe，并同步更新 `CHECKSUMS.txt` 和 `README.md`。
