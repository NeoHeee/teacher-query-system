# 师资信息查询系统 Windows 单文件版

这是一个用于单位内部师资信息整理、查询和管理的 Windows 单文件版程序。双击 `.exe` 即可启动，适合在普通 Windows 电脑上单机使用，也适合先做小范围试用。

## 下载使用

请下载仓库中的：

- `师资信息查询系统-Windows单文件版.exe`

下载后双击运行。程序启动后会自动打开浏览器页面。

默认登录账号：

- 账号：`admin`
- 密码：`admin123`

首次使用后建议立即进入用户管理修改管理员密码。

## 主要功能

- 师资信息查询
- 师资信息新增、编辑、批量管理
- Excel 数据导入
- 课题信息管理和查看
- 用户管理
- 数据库备份和恢复
- 操作日志
- 联系方式审核流程
- 标签/分类管理

基础版不影响日常使用。激活完整版后可使用师资合并、数据导出、附件档案等高级功能。

## 数据保存位置

程序第一次运行后，会在 exe 同级目录生成 `data` 文件夹。

重要数据主要保存在：

- `data/teacher_system.db`

请定期备份整个 `data` 文件夹，尤其是 `teacher_system.db` 数据库文件。

## 注意事项

- 请不要删除 `data` 文件夹，否则已有数据可能丢失。
- 请不要把真实数据库、附件、备份文件上传到公开网络。
- 本仓库只提供可公开下载的 Windows 单文件版程序和说明文档。
- 激活码生成器、私钥、真实业务数据不会上传到公开仓库。

## 适用场景

适合培训机构、高校、企事业单位内部建立师资资源库，用于统一维护讲师信息、讲授课题、联系方式、课酬评价和标签分类，减少多人反复整理 Excel 表格的工作。
